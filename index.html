install upk

bash -c "$(curl -fsSL https://raw.githubusercontent.com/juanvel4000/upk/HEAD/install.sh)"

packages

packages in upk are really simple. Just create a folder, then:


mkdir -p your-package/Info your-package/Source
# add your binaries to Source
# create Manifest.json in Info

upk build your-package
