Install upk

Just run this in your terminal

bash -c (curl -fsSL https://raw.githubusercontent.com/juanvel4000/upk/HEAD/install.sh)

Packages

Packages in upk are really simple, just create a folder, create folders `Info` and `Source` in it, then create a `Manifest.json` in `Info` and put your binaries in `Source`

Then run

upk build (folder)
